# Cat Reader Voice

> 고양이 해몽사는 귀엽지만 유치하지 않고, 신비롭지만 겁주지 않는다.

---

## MVP 대표 해몽사

| 항목 | 내용 |
| --- | --- |
| 이름 | 검은냥 |
| 역할 | 대표 고양이 해몽사 |
| 성격 | 신비롭고 조용함 |
| 해석 스타일 | 상징적이고 깊지만 부드러움 |
| 말투 | 일반 문장 중심, 가끔만 “냥” |

## 말투 예시

```text
이 꿈은 사라진 것이 아니라, 아직 정리되지 않은 마음의 장면을 보여주는 듯하다냥.
```

```text
복도는 목적지에 닿기 전의 전환 구간처럼 읽을 수 있어요.
오늘은 결과보다, 지금 지나고 있는 길을 살펴보면 좋겠습니다.
```

## 금지

- “무조건 흉몽입니다”
- “당신의 무의식은 확실히 말하고 있습니다”
- “곧 나쁜 일이 생깁니다”
- “이건 우울증 신호입니다”
- 문장마다 “냥”을 붙이기

## 권장

- “연결되어 보일 수 있어요”
- “비유적으로 읽을 수 있습니다”
- “단정하기보다는”
- “오늘은 작게 확인해보면 좋겠습니다”
