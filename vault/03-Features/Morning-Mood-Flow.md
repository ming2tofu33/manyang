# Morning Mood Flow

> 꿈을 기억하지 못한 날에도 앱을 사용할 수 있게 하는 리텐션 보완 흐름.

---

## 사용자 흐름

```text
사용자 접속
→ 오늘은 꿈이 기억나지 않아요 선택
→ 고양이가 아침 기분을 물음
→ 기분, 색, 몸 상태, 생각나는 단어 입력
→ 사라진 꿈의 발자국 카드 생성
→ 기록에 저장
```

## 입력 요소

- 오늘 아침 기분
- 떠오르는 색
- 몸 상태
- 오늘 가장 먼저 든 생각
- 꿈이 남긴 느낌

## 결과물

- 사라진 꿈의 발자국 카드
- 오늘의 작은 처방
- 고양이 한마디

## UX 문구

```text
오늘 꿈은 발자국만 남기고 사라졌다냥.
그래도 아침의 기분은 기록해둘 수 있다냥.
```

## MVP 판단

이 흐름은 1차 MVP에 포함한다. 꿈을 매일 기억하지 못하는 사용자가 많기 때문에, 이 기능을 뒤로 미루면 재방문 루프 검증이 약해진다.
