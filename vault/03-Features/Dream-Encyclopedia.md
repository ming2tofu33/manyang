# Dream Encyclopedia

> 마냥 꿈해몽이 단순 AI 해몽기가 아니라 자체 지식 베이스를 가진 서비스임을 보여주는 기능.

---

## 목적

사용자가 꿈을 입력하지 않아도 상징을 탐색할 수 있게 하고, 결과 화면의 해석 근거로 연결한다.

## 목록 화면

- 검색창
- 이번 주 자주 열린 상징
- 카테고리 필터: 장소, 사물, 사람, 자연, 감정, 행동
- 상징 카드 그리드
- 내 꿈에 나온 상징 표시

## 상세 화면

- 상징 이름
- 핵심 의미
- 긍정/부정 가능성
- 문맥 질문
- 관련 상징
- 내 꿈에 등장한 횟수
- 관련 꿈 기록

## MVP 범위

- seed 10개 상세 구현
- 30개까지 확장 가능한 구조
- 결과 화면에서 참고 항목 링크
