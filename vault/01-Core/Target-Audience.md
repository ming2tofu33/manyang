# Target Audience

> 꿈에 의미를 붙이고 싶지만, 무겁고 단정적인 해석은 부담스러운 사람.

---

## 1차 타깃

- 꿈을 자주 기억하거나 꿈에 의미를 부여하는 사람
- 감성적인 기록 앱을 좋아하는 사람
- 타로, 운세, 심리테스트, 다이어리 앱을 가볍게 즐기는 사람
- 귀여운 캐릭터 기반 서비스를 좋아하는 사람
- 자기 감정과 반복 패턴을 부드럽게 돌아보고 싶은 사람

## 2차 타깃

- AI 기반 감성 서비스를 좋아하는 사용자
- SNS에 예쁜 카드 결과를 공유하고 싶은 사용자
- 일기 앱은 부담스럽지만 짧은 기록은 남기고 싶은 사용자
- 고양이 캐릭터 IP를 좋아하는 사용자

## 핵심 동기

| 동기 | 제품 대응 |
| --- | --- |
| 어젯밤 꿈이 궁금하다 | 꿈 입력과 해몽 결과 |
| 꿈을 잊지 않고 저장하고 싶다 | 꿈 카드, 영수증, 달력 |
| 내 꿈에 자주 나오는 상징을 보고 싶다 | 상징 백과, 상징 히스토리 |
| 매일 아침 가벼운 루틴이 필요하다 | 기억 안 남/아침 기분 기록 |
| 귀여운 세계관이 좋다 | 고양이 해몽사, 달빛 냥몽관 |

## 리스크

- 꿈을 매일 기억하지 못하면 서비스 재방문이 약해질 수 있다.
- 해몽 결과가 너무 운세처럼 보이면 차별화가 약해진다.
- 고양이 말투가 과하면 유치하게 느껴질 수 있다.
