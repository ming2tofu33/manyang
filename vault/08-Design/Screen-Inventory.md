# Screen Inventory

> 먼저 핵심 루프 화면을 만들고, 백과/설정/꿈 씨앗은 우선순위에 따라 확장한다.

---

## 구현 방향이 잡힌 화면

- 홈 / 꿈 입력 진입 화면: 1차 구현 완료
- 꿈쓰기 화면: 1차 구현 완료
- 분석 중 로딩 화면: 1차 구현 완료
- 꿈 영수증 결과 화면: 1차 구현 완료
- 상징 백과 전체 화면: 1차 구현 완료
- 상징 백과 상세 화면: 복도: 1차 구현 완료
- 기록 / 꿈 달력 화면: 1차 구현 완료
- 기억나지 않아요 / 아침 기분 기록 화면: 1차 구현 완료
- 자기 전 꿈 씨앗 화면
- 설정 / 마이페이지 화면

## 추가로 필요한 화면

- 꿈 카드 / 꿈 영수증 저장 화면
- 꿈 카드 상세 화면
- 빈 기록 상태 화면
- 오류 / 분석 실패 화면
- 백과사전 검색 결과 없음 화면

## MVP 화면 우선순위

| 순위 | 화면 |
| --- | --- |
| 1 | 홈, 꿈 입력, 분석 중, 결과, 기록/달력 |
| 2 | 기억 안 남/아침 기분, 상징 백과 전체/상세 |
| 3 | 설정/마이페이지, 자기 전 꿈 씨앗 |

## 하단 내비게이션

현재 프론트 프로토타입은 참고 이미지 흐름에 맞춰 4개 탭으로 시작한다.

```text
오늘
꿈쓰기
기록
백과
```

확장 후:

```text
오늘
기록
백과
고양이방
마이
```
