# Access Plan Strategy

> 접근 정책은 비로그인 체험, 로그인 무료 루틴, 유료 상세 해몽의 3단 구조로 단순하게 시작한다.

---

## 결정 요약

마냥 꿈해몽의 접근/과금 구조는 다음 3단으로 둔다.

| 단계 | 대상 | 핵심 가치 | 제한/혜택 |
| --- | --- | --- | --- |
| Guest | 비로그인 사용자 | 오늘 한 번 꿈 영수증 체험 | 하루 1회 기본 해몽, 로컬 결과 중심 |
| Free Account | 로그인 무료 사용자 | 매일 기록되는 꿈 루틴 | 하루 1회 기본 해몽, 영수증 보관, 발자국/씨앗 |
| Moon Pass | 유료 사용자 | 깊은 해몽과 누적 분석 | 상세 해몽, 회색냥, 타로 상징 리딩, 월간 정산 |

핵심은 기본 경험을 너무 일찍 막지 않는 것이다. 첫 꿈 해몽과 기본 꿈 영수증은 무료로 충분히 경험하게 하고, 유료화는 **더 깊은 해석, 더 오래 남는 기록, 더 특별한 해몽사**에서 시작한다.

## 단계별 정책

### Guest

비로그인 사용자는 로그인 장벽 없이 오늘의 꿈 해몽을 한 번 받을 수 있다.

제공:

- 하루 1회 기본 꿈 해몽
- 기본 꿈 영수증 보기
- 기본 저장/공유 또는 이미지 다운로드
- 검은냥 기본 해몽

제한:

- 서버 보관 없음
- 기기/브라우저 기반 1일 제한
- 기록/달력/상징 히스토리는 제한적으로만 노출
- 다음날 이어서 보려면 로그인 유도

목적:

- 검색/SNS 유입 사용자의 첫 경험 마찰을 낮춘다.
- "꿈 영수증이 저장하고 싶은가"를 빠르게 검증한다.
- 가입 전 제품 가치를 보여준다.

권장 CTA:

```text
오늘의 꿈 영수증을 받았어요.
로그인하면 내 꿈 기록에 매일 쌓아둘 수 있어요.
```

### Free Account

로그인 무료 사용자는 매일 기본 꿈 해몽을 받을 수 있고, 꿈 영수증과 발자국이 계정에 쌓인다.

제공:

- 하루 1회 기본 꿈 해몽
- 검은냥, 하얀냥, 치즈냥 무료 선택
- 꿈 영수증 보관
- 꿈 기록/달력
- 아침 기분 기록
- 발자국 기록
- 하루 1개 꿈 씨앗
- 기본 상징 백과 열람

제한:

- 상세 해몽 제한
- 회색냥 잠금
- 타로 상징 리딩 잠금
- 월간 꿈 정산서 잠금 또는 미리보기만 제공
- 심화 상징 히스토리 잠금

목적:

- 매일 들어오는 습관을 만든다.
- 무료 고양이 3마리의 차이를 체감하게 한다.
- 기록이 쌓인 뒤 Moon Pass 전환 타이밍을 만든다.

권장 CTA:

```text
오늘의 꿈은 기록에 남았어요.
7일이 쌓이면 이번 주 꿈 흐름을 살짝 보여드릴게요.
```

### Moon Pass

Moon Pass는 기본 해몽을 더 많이 파는 플랜이 아니라, 누적 기록을 더 깊고 예쁘게 읽는 플랜이다.

제공:

- 상세 꿈 해몽
- 회색냥 꿈+타로 상징 리딩
- 월간 꿈 정산서
- 상징 히스토리 심화 보기
- 고급 꿈 영수증 템플릿
- 고해상도/PDF 저장
- 무제한 또는 확장 꿈 영수증 보관함
- 검색/필터 고급 기능
- 프라이버시 잠금

목적:

- "내 꿈 세계가 쌓였다"는 사용자가 더 깊은 분석에 돈을 내게 한다.
- 회색냥을 연간 구독의 대표 얼굴로 사용한다.
- 월간 정산서로 반복 결제의 이유를 만든다.

권장 CTA:

```text
회색냥이 지난 꿈 기록과 오늘의 타로 상징을 함께 읽어드려요.
Moon Pass로 깊은 꿈 해몽을 열어보세요.
```

## 사용량 제한 원칙

MVP 사용량 제한은 정교한 결제 시스템보다 명확한 제품 경험을 우선한다.

| 항목 | Guest | Free Account | Moon Pass |
| --- | --- | --- | --- |
| 기본 꿈 해몽 | 1일 1회 | 1일 1회 | 1일 1회 이상 또는 정책 확정 전까지 동일 |
| 기본 꿈 영수증 | 가능 | 가능 | 가능 |
| 기록 보관 | 로컬/임시 | 계정 보관 | 확장 보관 |
| 무료 고양이 3마리 | 검은냥 우선 | 검은냥/하얀냥/치즈냥 | 검은냥/하얀냥/치즈냥 |
| 상세 해몽 | 불가 | 관심 버튼 | 가능 |
| 회색냥 | 잠금 안내 | 잠금 안내 | 가능 |
| 월간 정산서 | 불가 | 미리보기/관심 버튼 | 가능 |

주의:

- Guest 1일 제한은 완벽한 보안보다 남용 방지 수준으로 시작한다.
- IP만으로 제한하면 가족/카페/회사 네트워크에서 오탐이 생길 수 있다.
- MVP에서는 브라우저 localStorage + 서버 rate limit 조합을 우선 검토한다.
- 계정 기반 저장은 Supabase Auth 도입 이후 구현한다.

## 전환 타이밍

프리미엄 제안은 사용자가 가치를 느낀 뒤에만 보여준다.

| 순간 | 제안 |
| --- | --- |
| Guest 첫 영수증 확인 후 | 로그인하면 매일 기록 가능 |
| 로그인 후 첫 기록 저장 | 내 꿈 기록이 쌓이기 시작했다는 메시지 |
| 기록 3개 이상 | 상징 히스토리 미리보기 |
| 기록 7개 이상 | 월간/주간 흐름 미리보기 |
| 월말 | 월간 꿈 정산서 제안 |
| 반복 상징 발견 | 회색냥 깊은 분석 제안 |

피해야 할 것:

- 첫 화면에서 구독부터 요구하기
- 기본 꿈 해몽을 바로 잠그기
- 회색냥의 차이를 체감하기 전에 결제만 요구하기
- 무료 사용자가 저장한 기록 접근을 갑자기 막기

## 상세 해몽 정의

유료 상세 해몽은 기본 해몽보다 길기만 한 결과가 아니다.

기본 해몽:

- 꿈 요약
- 주요 상징 3~5개
- 백과사전 기반 짧은 리딩
- 오늘의 작은 처방
- 기본 꿈 영수증

상세 해몽:

- 상징별 세부 해석
- 감정 흐름 설명
- 반복 상징 연결
- 내 기록과의 비교
- 회색냥 코멘트
- 타로 상징 리딩
- 오늘의 질문
- 고급 꿈 영수증 또는 월간 정산 연결

## 구현 우선순위

1. 접근 정책을 코드와 문서에서 고정한다.
2. Guest/Free/Paid 상태를 표현하는 타입을 만든다.
3. 하루 1회 기본 해몽 제한을 구현한다.
4. 로그인 유도 CTA를 결과 화면에 넣는다.
5. 로그인 무료 사용자 저장 루프를 구현한다.
6. 상세 해몽 CTA를 달조각/회색냥 관심 버튼으로 먼저 검증한다.
7. 월간 정산서와 회색냥을 실제 유료화 후보로 묶는다.
