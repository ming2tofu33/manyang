# Pricing Revenue Strategy

> 가격 전략은 기본 해몽을 막아서 돈을 받는 구조가 아니라, 깊은 해석과 기록 자산, 회색냥의 특별한 경험에 돈을 받는 구조로 시작한다.

---

## 결정 요약

마냥 꿈해몽의 1차 수익 모델은 **Moon Pass 구독**으로 둔다.

추천 시작 가격:

| 상품 | 권장 가격 | 역할 |
| --- | ---: | --- |
| Moon Pass 월간 | 5,900원/월 | 진입 장벽이 낮은 유료 체험 |
| Moon Pass 연간 | 49,000원/년 | 핵심 구독 상품 |
| 초기 사용자 연간가 | 39,000원/년 | 출시 초반 전환 검증용 |
| 상세 해몽 1회권 | 1,500~2,000원 | 구독 전 맛보기 |
| 월간 꿈 정산서 단건 | 3,900원 | 리포트 수요 검증 |
| 프리미엄 영수증/PDF | 1,100~2,200원 | 결과물 소장 가치 검증 |

MVP에서는 실제 결제보다 **관심 클릭과 전환 의향 측정**을 먼저 한다. 결제는 사용자가 어떤 프리미엄 기능에 반응하는지 확인한 뒤 붙인다.

## 포지셔닝

마냥은 단순한 "AI 꿈해몽 생성기"보다 **꿈을 기록하고 정산하는 감성 아카이브**에 가깝게 포지셔닝한다.

무료 경험의 핵심:

- 로그인 없이 하루 1회 기본 꿈 해몽
- 기본 꿈 영수증
- 검은냥, 하얀냥, 치즈냥의 기본 해몽 톤
- 로그인 후 매일 기록과 보관

유료 경험의 핵심:

- 상세 꿈 해몽
- 회색냥 꿈+타로 상징 리딩
- 월간 꿈 정산서
- 상징 히스토리 심화 보기
- 고급 꿈 영수증 템플릿
- 고해상도/PDF 내보내기

돈을 받는 이유는 "더 많이 해몽해준다"가 아니라 **내 꿈 기록이 쌓였을 때 더 깊고 예쁘게 읽어준다**여야 한다.

## 수익원 설계

### 1. Moon Pass

Moon Pass는 메인 수익원이다.

포함할 기능:

- 모든 고양이의 심화 꿈해몽
- 회색냥 전용 꿈+타로 상징 리딩
- 월간 꿈 정산서
- 반복 상징 분석
- 고급 꿈 영수증 테마
- 고해상도 이미지/PDF 저장
- 확장 보관함

권장 메시지:

```text
회색냥이 지난 꿈 기록과 오늘의 타로 상징을 함께 읽어드려요.
Moon Pass로 깊은 꿈 해몽을 열어보세요.
```

### 2. 상세 해몽 1회권

구독 전 결제 장벽을 낮추기 위한 단건 상품이다.

용도:

- Moon Pass 결제가 부담스러운 사용자에게 테스트 기회 제공
- 상세 해몽 가격 민감도 확인
- 구독 전환 전 맛보기 역할

권장 가격:

- 1,500원
- 2,000원

주의:

- 1회권이 너무 강하면 구독 전환을 해칠 수 있다.
- 회색냥 월간 리포트, 누적 분석, 고급 보관함은 구독에 남겨둔다.

### 3. 월간 꿈 정산서

구독의 가장 강한 명분이 될 수 있는 상품이다.

구성:

- 이번 달 꿈 횟수
- 자주 나온 상징
- 감정 흐름
- 반복된 장소/인물/상황
- 회색냥 코멘트
- 다음 달 질문
- 저장 가능한 월간 영수증/PDF

권장 방식:

- Moon Pass에는 포함
- Free Account에는 미리보기 제공
- 단건 구매는 3,900원으로 테스트

### 4. 프리미엄 꿈 영수증

꿈 영수증은 마냥의 대표 결과물이므로, 예쁜 저장물에 대한 지불 의향을 확인할 수 있다.

과금 후보:

- 고급 영수증 테마
- 고해상도 이미지 저장
- PDF 내보내기
- 월간 합본 영수증
- 특별 시즌 테마

MVP에서는 실제 결제보다 `프리미엄 영수증 만들기` 관심 버튼으로 먼저 검증한다.

### 5. 광고

강제 광고는 MVP 수익원으로 권장하지 않는다.

이유:

- 꿈 기록은 사적인 경험이어서 광고가 몰입을 깨기 쉽다.
- 운세/상담형 서비스에서 광고가 많으면 신뢰가 낮아질 수 있다.
- 초기에는 브랜드 감성과 반복 사용이 더 중요하다.

광고를 넣는다면:

- 선택형 리워드 광고
- 무료 사용자의 추가 프리미엄 영수증 1회
- 이벤트성으로만 테스트

| PDF/고해상도 저장 | 제한 | 단건 후보 | 포함 |

## MVP 검증 이벤트

결제 전에는 다음 이벤트를 먼저 심는다.

| 이벤트 | 검증 질문 |
| --- | --- |
| `detailed_reading_interest_clicked` | 상세 해몽에 돈을 낼 의향이 있는가 |
| `gray_cat_interest_clicked` | 회색냥이 프리미엄 캐릭터로 작동하는가 |
| `monthly_report_interest_clicked` | 월간 정산서가 구독 이유가 되는가 |
| `premium_receipt_interest_clicked` | 예쁜 저장물에 결제 의향이 있는가 |
| `guest_login_cta_clicked` | 첫 영수증 후 로그인 전환 가능성이 있는가 |
| `guest_daily_limit_hit` | 하루 1회 제한이 가입 동기를 만드는가 |

## 가격 판단 기준

가격을 올릴 수 있는 조건:

- 월간 꿈 정산서 클릭률이 높다.
- 회색냥 관심 클릭률이 높다.
- 7개 이상 기록한 사용자의 재방문율이 높다.
- 사용자가 영수증을 저장/공유한다.
- 사용자가 "내 기록과 연결된 해몽"을 원한다.

가격을 낮춰야 하는 조건:

- 상세 해몽 관심 클릭은 높지만 결제 의향이 낮다.
- 사용자가 구독보다 단건 구매만 원한다.
- 기록 누적 전에 이탈이 많다.
- 회색냥과 무료 고양이의 차이를 체감하지 못한다.

## 비용 관리 원칙

AI 비용은 모델 가격과 토큰 사용량에 따라 변한다. 결제 출시 전에는 [OpenAI API Pricing](https://platform.openai.com/docs/pricing)을 다시 확인한다.

운영 원칙:

- 기본 해몽은 짧고 안정적인 구조로 유지한다.
- 상세 해몽은 Moon Pass 또는 단건 결제 뒤에만 생성한다.
- 월간 정산서는 충분한 기록이 있을 때만 생성한다.
- 같은 꿈에 대한 재요청은 캐시 또는 저장된 결과를 우선 사용한다.
- 타로 리딩은 1장 상징 리딩으로 시작해 토큰 비용을 제한한다.

## 피해야 할 과금

- 기본 꿈 기록 저장을 갑자기 유료화하기
- 무료 사용자가 이미 저장한 기록을 잠그기
- 꿈 해몽을 확률형 뽑기처럼 만들기
- 타로 카드를 반복 뽑게 하며 과금 유도하기
- 불안감을 자극해 결제시키기
- 발자국을 현금으로 판매하기
- 씨앗을 꿈 조작 아이템처럼 표현하기

## 실행 순서

1. Guest/Free Account/Moon Pass 접근 정책을 코드에 고정한다.
2. 하루 1회 기본 해몽 제한을 구현한다.
3. 결과 화면에 로그인 CTA를 넣는다.
4. 결과 화면에 상세 해몽 관심 CTA를 넣는다.
5. 회색냥 잠금/관심 CTA를 넣는다.
6. 월간 꿈 정산서 미리보기 버튼을 넣는다.
7. 프리미엄 영수증 관심 버튼을 넣는다.
8. 이벤트 클릭률을 보고 결제 우선순위를 정한다.
9. 결제는 Moon Pass 연간가와 상세 해몽 1회권부터 테스트한다.
