# KPI Gates

> MVP 성공은 “만들었는가”가 아니라 “다시 쓰고 싶게 만들었는가”로 판단한다.

---

## 정량 지표

- 첫 방문 후 꿈 입력 완료율
- 결과 저장률
- 다음날 재방문율
- 꿈 카드 공유 클릭률
- 평균 꿈 입력 길이
- 기억 안 남 흐름 사용률

## 정성 지표

- 해몽 결과가 그럴듯한가?
- 카드나 영수증이 저장하고 싶은가?
- 고양이 테마/안내 말투가 해몽 신뢰를 해치지 않는가?
- 운세 앱처럼 흔하게 느껴지지 않는가?
- 백과사전 근거가 신뢰감을 주는가?

## Gate 기준 초안

| Gate | 통과 기준 |
| --- | --- |
| G1 입력 | 테스트 사용자 5명 중 4명이 꿈 입력 완료 |
| G2 결과 | 테스트 사용자 5명 중 3명이 결과가 흥미롭다고 평가 |
| G3 저장 | 테스트 사용자 5명 중 3명이 카드/영수증 저장 의향 |
| G4 재방문 | 다음날 다시 기록하고 싶다는 응답 50% 이상 |
