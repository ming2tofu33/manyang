# Retention Loops

> 재방문은 해몽 자체보다 “기록이 쌓이는 감각”에서 나온다.

---

## 아침 루프

```text
꿈이 기억나는지 확인
→ 기억나면 꿈 입력
→ 기억나지 않으면 아침 기분 기록
→ 오늘의 작은 처방 확인
→ 꿈 카드 또는 사라진 꿈 카드 저장
```

## 밤 루프

자기 전 루틴은 꿈을 유도하는 의도 선택보다, 오늘 하루의 기분과 컨디션을 남기는 밤의 기록으로 가져간다.

```text
자기 전 접속
→ 오늘 하루의 기분과 몸 컨디션 선택
→ 한 줄 기록 남기기
→ 다음날 꿈 해몽에서 어젯밤의 맥락으로 참고
```

## 핵심 리텐션 장치

| 장치 | MVP 포함 | 이유 |
| --- | --- | --- |
| 꿈 달력 | 예 | 기록이 쌓이는 감각 |
| 기억 안 남 흐름 | 예 | 매일 꿈을 기억하지 못하는 문제 보완 |
| 상징 히스토리 | 부분 | 자주 등장한 상징을 보여주는 재미 |
| 주간 리포트 | 아니오 | 2차 MVP |
| 밤의 기록 | 예 | 해몽 신뢰도와 다음날 재방문 보강 |
