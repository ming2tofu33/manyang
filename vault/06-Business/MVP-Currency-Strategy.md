# MVP Currency Strategy

> MVP의 재화 시스템은 과금 완성이 아니라, 재방문과 유료 의향을 검증하기 위한 최소 루프다.

---

## 핵심 방향

마냥 꿈해몽의 유료화는 **더 많은 해몽 판매**보다 **꿈이 쌓였을 때 생기는 개인 아카이브, 영수증, 상징 히스토리, 월간 정산 가치**에 초점을 둔다.

MVP에서는 실제 결제나 복잡한 상점보다 다음 세 가지를 검증한다.

| 검증할 질문 | MVP 장치 |
| --- | --- |
| 꿈이 없어도 매일 다시 들어오는가? | 발자국 기록 |
| 밤에 심은 행동이 다음날 재방문을 만드는가? | 꿈 씨앗 |
| 사용자가 더 깊은 해몽이나 예쁜 결과물에 관심을 보이는가? | 달조각 버튼 |

## 결과물 우선순위

MVP의 메인 결과물은 **꿈 카드**보다 **꿈 영수증**으로 둔다.

카드는 타로, 오라클, 운세 앱에서 익숙한 형식이라 뻔하게 보일 수 있다. 반면 꿈 영수증은 꿈속 상징과 감정을 항목처럼 정리하고, "어젯밤 꿈에서 무엇을 가져왔는지 정산한다"는 마냥만의 인상을 만들 수 있다.

카드는 버리지 않고, 이후 특별 이벤트, 월간 리포트, 고급 테마, 공유용 보조 결과물로 확장한다.

## MVP 재화 3종

### 발자국

**역할:** 매일 남긴 흔적, 아침 루틴, 기록 누적감.

발자국은 현금 구매 재화가 아니라 사용자가 앱에 들어와 기록하면서 쌓는 활동 재화다. 돈으로 살 수 있게 하면 "내가 남긴 흔적"이라는 의미가 약해지므로 MVP에서는 구매 불가로 둔다.

얻는 방법:

- 아침 기분 기록
- 꿈이 기억나지 않는 날의 발자국 기록
- 꿈 해몽 후 영수증 저장
- 연속 기록 유지

MVP 사용처:

- 달력에 발자국 표시
- 이번 달 발자국 수 표시
- 일정 개수 달성에 따른 추가 혜택 없음

MVP 예시:

- 발자국은 달력 표시, 이번 달 수, streak 유지에만 사용한다.
- 개수 달성에 따른 추가 혜택은 MVP에서 제외한다.

### 씨앗

**역할:** 밤 루틴, 다음날 재방문 약속.

씨앗은 상점 재화보다 자기 전 루틴 아이템으로 둔다. 사용자가 밤에 "오늘 밤 마음에 심어둘 질문"을 고르고, 다음날 아침 기록으로 돌아오게 만드는 장치다.

MVP 사용 방식:

- 하루 1개 무료 씨앗
- 밤에 씨앗 심기
- 다음날 홈 또는 아침 화면에 어제 심은 씨앗 표시
- 아침 기록 완료 시 씨앗이 새싹 기록 또는 발자국 보너스로 전환

씨앗 예시:

| 씨앗 | 의미 |
| --- | --- |
| 기억 씨앗 | 내일 아침 꿈을 붙잡기 위한 질문 |
| 평온 씨앗 | 잠들기 전 부드러운 문장 |
| 용기 씨앗 | 다음날 받고 싶은 작은 처방 방향 |
| 상징 씨앗 | 특정 상징 백과와 연결된 질문 |

주의할 점:

- 씨앗이 꿈을 조작한다고 말하지 않는다.
- "내일 아침 기억을 붙잡기 위한 작은 씨앗"처럼 부드럽게 표현한다.

### 달조각

**역할:** 프리미엄 의향 검증, 심화 기능의 열쇠.

달조각은 MVP에서 실제 결제 재화가 아니다. "나중에 돈을 받을 기능이 무엇인지"를 확인하는 프리미엄 재화의 껍데기로 사용한다.

MVP 사용 방식:

- 기본 해몽, 기본 영수증, 기본 저장은 무료로 제공한다.
- 결과 화면과 기록 화면에 달조각 기반 프리미엄 버튼을 노출한다.
- 클릭 시 결제 대신 관심 표시, 준비 중 안내, 알림 신청 등으로 처리한다.
- 클릭률로 유료화 우선순위를 판단한다.

MVP 버튼 예시:

| 버튼 | 목적 |
| --- | --- |
| 달조각 2개로 심화 해몽 보기 | 심화 해몽 수요 확인 |
| 달조각 3개로 고급 꿈 영수증 만들기 | 예쁜 결과물 수요 확인 |
| 달조각 5개로 월간 꿈 정산서 미리보기 | 리포트 수요 확인 |
| 달조각으로 상징 히스토리 열기 | 누적 분석 수요 확인 |

나중에 실제 유료화 단계에서 달조각으로 열 수 있는 것:

- 추가 해몽
- 심화 상징 리딩
- 고급 꿈 영수증 템플릿
- 고해상도 저장
- 월간 꿈 정산서
- 특별 고양이 해몽사
- 상징 히스토리 심화 보기

## MVP 루프

```text
아침
꿈 기억남 → 꿈 입력 → 꿈 영수증 받기 → 기록 저장 → 발자국 +1

아침
꿈 기억 안 남 → 기분 기록 → 사라진 꿈 발자국 남김 → 발자국 +1

밤
꿈 씨앗 심기 → 다음날 홈에서 씨앗 확인 → 아침 기록 유도

결과 화면
기본 꿈 영수증 제공 → 달조각 프리미엄 버튼 노출 → 클릭률 측정
```

## MVP에 넣을 비즈니스 요소

- 기록 화면 상단에 `꿈 기록`, `발자국 기록`, `꿈 씨앗`, `이번 달 상징` 수를 보여준다.
- 결과 화면의 메인 결과물은 꿈 영수증으로 둔다.
- 결과 화면 하단에 달조각 기반 심화 기능 버튼을 둔다.
- 밤 화면에서 하루 1개 씨앗을 심을 수 있게 한다.
- 아침 화면에서 어제 심은 씨앗을 회수하거나 기록으로 이어지게 한다.
- 발자국은 기록 흔적으로만 사용한다.

## MVP에서 하지 않을 것

- 실제 현금 결제
- 달조각 현금 구매
- 복잡한 상점
- 발자국으로 핵심 기능 잠금
- 기본 꿈 기록 유료화
- 기본 꿈 영수증 유료화
- 기본 저장/삭제/프라이버시 기능 유료화
- 확률형 뽑기처럼 보이는 보상

## 측정 지표

| 지표 | 의미 |
| --- | --- |
| 아침 기분 기록률 | 꿈이 없는 날에도 루틴이 작동하는가 |
| 씨앗 심기 클릭률 | 밤 루틴에 관심이 있는가 |
| 씨앗 심은 다음날 재방문율 | 씨앗이 재방문을 만드는가 |
| 발자국 3개 이상 사용자 비율 | 반복 사용자가 생기는가 |
| 꿈 영수증 저장률 | 결과물이 소장 가치가 있는가 |
| 달조각 버튼 클릭률 | 프리미엄 기능 의향이 있는가 |
| 월간 정산서 관심 클릭률 | 구독/리포트 수요가 있는가 |

## MVP 성공 기준 초안

테스트 사용자 10명 기준:

- 6명 이상이 꿈 또는 아침 기분 기록 완료
- 4명 이상이 꿈 영수증을 저장하고 싶다고 응답
- 3명 이상이 씨앗을 심고 다음날 다시 접속
- 2명 이상이 달조각 기반 심화 기능 버튼 클릭

## Post-MVP 유료화 방향

실제 유료화는 달조각과 Moon Pass를 중심으로 검토한다.

| 유료화 후보 | 형태 |
| --- | --- |
| 무제한 꿈 영수증 보관함 | Moon Pass |
| 월간 꿈 정산서 | Moon Pass 또는 단건 결제 |
| 상징 히스토리 심화 보기 | Moon Pass |
| 고급 꿈 영수증 템플릿 | 달조각 또는 단건 결제 |
| 고해상도/PDF 저장 | 달조각 또는 단건 결제 |
| 특별 고양이 해몽사 | Moon Pass 또는 단건 결제 |
| 추가 해몽/심화 해몽 | 달조각 |
